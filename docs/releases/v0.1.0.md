# v0.1.0 发布说明

发布日期：2026-02-23

## 概览
本版本为项目首个可发布版本，完成了从流程规范、数据引擎、FDS 执行到蒸馏训练与推理基准的端到端 MVP 能力。

## 主要内容
1. 研发流程与规范
- 建立 Git Flow 与 Issue 驱动开发规范。
- 增加 issue 模板、流程文档与分支创建脚本。

2. 数据与仿真流水线
- 工况采样与任务 manifest 生成。
- 自动生成最小 FDS 输入文件。
- 批量执行器、运行摘要与端到端流水线脚本。

3. 评估与蒸馏
- 评估指标脚本（MAE/NMAE/Accuracy/P95）。
- 蒸馏训练 MVP（线性与 MLP）。
- 推理接入与 P95 基准测试（毫秒级）。

4. 真实 FDS 标签链路
- 从真实 `run.log` 提取标签。
- 标签质量门控与真实标签蒸馏训练闭环。

## 当前已知限制
- GPU 当前处于异常需复位状态，需重启后再做 GPU 训练复检。
- 真实标签蒸馏验证集精度尚未达到目标 85%，后续需继续扩样本与提升标签质量。
