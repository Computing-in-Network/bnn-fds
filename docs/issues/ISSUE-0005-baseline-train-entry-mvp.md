# ISSUE-0005 基线训练入口 MVP（占位可运行）

## 状态
- Done

## 背景
当前已具备数据工况清单、任务 manifest 和评估脚本，需要补齐训练入口骨架，形成“可运行的端到端最小链路”。

## 目标
- 提供训练入口脚本 `train/train_mvp.py`。
- 支持读取配置与输入清单。
- 输出训练占位结果（模型元信息与日志），为后续真实训练替换做好接口。

## 验收标准
- 可执行命令：`python3 train/train_mvp.py --config configs/train/train_mvp.json`。
- 输出目录：`train/artifacts/mvp_run/`。
- 产出文件至少包含：`model_meta.json`、`train_log.json`。
- 文档说明齐全（输入、输出、命令示例）。

## 范围
- In Scope：
  - 训练入口与参数解析。
  - 占位训练流程与产物落盘。
- Out of Scope：
  - 真实神经网络训练与收敛。
  - GPU 分布式训练。

## 风险
- 若接口定义不稳定，后续替换真实训练器成本上升。
- 配置字段不规范会导致后续链路重复改造。
